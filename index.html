What is ModMaker?

ModMaker is a Java aplication desinged to make it easy to create a Minecraft mod.

UI

ModMaker's UI is desinged to make it easy for someone to make a mod.

The image above shows the main ModMaker screen with the options new, save, open, export, mod info, and about. The "New" button creates a new mod and pop up the mod info dialog. "Save" and "Open" save/open ModMaker (.mcm) files. "Export" export Java source code that still needs to be compiled. "Mod Info" opens up a dialog to edit information about the mod ie(name, discription, author). "About" opens the about screen.
